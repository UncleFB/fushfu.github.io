Hugo

过年好！新的一年从搭个博客开始。 Hugo 简介 Hugo主页 我了解的博客框架不多，Hugo 的主页上赫然写到 The world’s fastest framework for building websites，感觉很牛的样子。 Hugo 是使用 Go 语言编写的，是一个快速的现代静态页面生成器，它让建网页再次有趣 (make website creation fun again)。 Hugo 安装 我这边是用了 WSL2，所以可以按照官网上 Ubantu 系统的方式来安装 可以直接执行 sudo apt install hugo 来完成安装。 新建一个网站 hugo new site fushfu cd fushfu git init git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke themes/ananke echo "theme = 'ananke'" >> config.toml hugo server 通过上面的指令，新建一个网站，并且使用了 ananke 主题。 执行 hugo server 后，会开启服务，访问 localhost:1313 就可以看到你的网站了。
