Hugo 简介

Hugo主页

我了解的博客框架不多，Hugo 的主页上赫然写到 The world’s fastest framework for building websites，感觉很牛的样子。

Hugo 是使用 Go 语言编写的，是一个快速的现代静态页面生成器，它让建网页再次有趣 (make website creation fun again)。

Hugo 安装

我这边是用了 WSL2，所以可以按照官网上 Ubantu 系统的方式来安装

可以直接执行 sudo apt install hugo 来完成安装。

新建一个网站

1
2
3
4

hugo new site fushfu
cd fushfu
git init
git submodule add https://github.com/olOwOlo/hugo-theme-even themes/even

这里我选择了 Even 这个主题。进到 themes/even/exampleSite 文件夹下，复制 config.toml 到 项目根目录下。执行 hugo server，会启动服务，之后打开 localhost:1313

执行 hugo server 后，会开启服务，访问 localhost:1313 就可以看到你的网站了。

新建一个网页

执行 hugo new post/hugo.md 就创建了你现在看到的页面。

编译部署

一般页面头部会有一个 draft 属性，如果是 true， 则默认编译的时候不会输出这个页面。当内容写完之后，就可以将 draft 设置为 false。然后执行 hugo new post/hugo.md

执行 hugo 就可以构建静态网站了，可以使用 destination 参数指定编译文件的输出位置。

然后可以部署在 githubpage 上，具体方法就是在 github 上创建一个和 github 用户名相同的仓库，然后把编译好的项目推上去，访问 username.github.io (username是你的 github 用户名)，即可访问、
